## Pro Tips

💡 **Bookmark these files:**
- [QUICKSTART.md](./QUICKSTART.md) - Most frequently used
- [TROUBLESHOOTING.md](./TROUBLESHOOTING.md) - For when things break
- [DEVELOPMENT.md](./DEVELOPMENT.md) - For coding sessions

💡 **Browser search (Ctrl+F):**
- Search documentation files for specific topics
- Example: Search README.md for "validation"

💡 **Local server reference:**
- Backend: http://localhost:5000
- Frontend: http://localhost:3000
- Health check: http://localhost:5000/api/health

💡 **Quick restart:**
If something's wrong, the fix is usually:
1. Kill all Node processes
2. Clear npm cache: `npm cache clean --force`
3. Reinstall: `npm install`
4. Restart: `npm start`
